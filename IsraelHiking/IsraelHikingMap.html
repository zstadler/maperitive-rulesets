Israel Hiking Project

This map was generated from Open Street Map (OSM) data where it can be be viewed and edited. All you need to do is create an account and start mapping.

The map can also be downloaded and used offline on Android devices. You'll need to install OruxMaps and place the folder "Israel Hiking" with the two files on your SD card under
/sdcard/oruxmaps/mapfiles/

Below are some links related to this project

  • See Github for files and instructions on how to create the map by yourself
  • See Israel OSM Wiki Project for common rules of the Israeli Open Street Map community
  • See Israel OSM Forum for the Israeli Open Street Map forum where you can post questions and look for answers

Please consider joining Dropbox using this link. Both you and this project will receive a 500Mb free storage bonus.

פרוייקט מפת הטיולים הפתוחה של ישראל

המפה הזו נוצרה בעזרת מידע מ- Open Street Map שם ניתן לראות ולערוך אותו. כל שיש לעשות הוא לפתוח חשבון ולהתחיל למפות.

המפה ניתנת להורדה ולשימוש בצורה לא מקוונת על מכשירי אנדרואיד. יש להתקין את OruxMaps ולשים את התיקיה "Israel Hiking" עם שני הקבצים שלה על כרטיס הזיכרון בתיקיה
/sdcard/oruxmaps/mapfiles/
